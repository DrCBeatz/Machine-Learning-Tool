How to use this app

This app lets you upload and use custom datasets (or use built-in datasets popular in machine learning), train and test these models using various binary classification algorithms (select 'Algorithms' from the top menu to learn more) and save their results (known as entries). You can then create graphs comparing the train and test accuracy of a particular dataset, algorithm and model parameter.

How to train and test a model for a built-in dataset

The models that you train and test are known as entries. To create an entry:

  1. Select ‘Add Entry’ from the top menu.
  2. Under ML type (the type of algorithm), select ‘binary classification’.
  3. Select a dataset (try the Wisconsin Breast Cancer dataset to start).
  4. Select an algorithm (try K-Nearest Neighbours).
  5. Enter a number for parameter 1 value (this can be any positive integer value, choose a number between 1 and 10).
  6. Click or tap the ‘submit’ button.

You will then be taken to the ‘Entries’ page, which shows a list of all the models you have trained and tested, and the train/test accuracy results (which is represented by a decimal number from 0 to 1, where 1 equals 100% accuracy). From here you can either add another entry, edit or delete any entries you’ve made.

How to create a graph to plot the train/test accuracy of an algorithm & dataset

  1. Add at least 3 entries for the same dataset, algorithm, and parameter number (which is 1 by default).
  2. Select ‘Add Graph’ from the top menu.
  3. Choose the ML type, dataset, algorithm, and parameter to plot (1 by default on most algorithms, i.e. enter the number 1).
  4. Click or tap the ‘submit’ button.

You will then be taken to the Graphs page, where you should see the title of the graph you just created. Click on the title of the graph to view the graph.

How to upload a custom dataset
  1. Select ‘Add Dataset’ from the top menu.
  2. Enter and title and description of the dataset.
  3. Under ML Type, select ‘binary classification’.
  4. Click or tap the ‘upload file’ and locate the file on your hard drive.
  5. Select the filetype (formats supported include CSV, Excel .xlsx and .xls)*.
  6. Click or tap the ‘submit’ button.

You will be taken to the ‘Datasets’ page, and you should now see your dataset added to the list. You can then make entries and graphs using this custom data set. Note that if you delete this dataset, any entries or graphs you make using this dataset will be deleted also.

*(for datasets, the last column will be the target or output variable, and by default Excel files will ignore the first row of the dataset as this is header data, usually the title, which is not needed during training/testing and is provided by the user when uploading the dataset)
